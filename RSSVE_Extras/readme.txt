==================================================
Real Solar System Visual Enhancements (RSSVE)
==================================================

High Resolution (HR) cloud pack:

RSSVE offers an optional 32K high resolution cloud pack (Earth only for now). To
install it just copy or move the contents of the "RSSVE_Extras" folder inside the
GameData folder. You will be asked to replace the following six (6) files:

    • EarthCloudsXn
    • EarthCloudsXp
    • EarthCloudsYn
    • EarthCloudsXp
    • EarthCloudsZn
    • EarthCloudsZp

Replace the files and then (re)start KSP. That's it!

Note that the high resolution cloud pack requires at least 16 GB of system RAM
and 4 GB of VRAM.
